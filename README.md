# SubAudit

Subscription tracker and auditor for Android.

## Features

- 📊 Dashboard with monthly/yearly cost overview
- 📅 Upcoming payment tracking (7/30 days)
- 🏷️ Category-based breakdown
- ⏸️ Mark subscriptions as active/paused/cancelled
- 💷 Multi-currency support (GBP, USD, EUR)
- 🔔 Renewal reminders (notification before payment)
- 🔍 Duplicate detection
- 📤 Export/import subscription data as JSON
- 🌙 Dark theme, Material Design 3
- ℹ️ About section with update check & share

## Built With

- Kotlin + AndroidX
- Room database
- Material Design 3
- AlarmManager for reminders
- Coroutines (Dispatchers.IO for all DB ops)
